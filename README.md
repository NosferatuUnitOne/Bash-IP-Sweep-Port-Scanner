# Bash IP Sweeper + Nmap Scanner

A simple bash script that performs basic network reconnaissance.

## Features

- Ping sweeps a /24 network
- Collects alive hosts
- Automatically scans live hosts with Nmap

## Usage

./ipsweep.sh 192.168.1

Example:

./ipsweep.sh 192.168.1

## Requirements

- Bash
- Nmap
